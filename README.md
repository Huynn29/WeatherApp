# WeatherApp 🌦️

Ứng dụng Android xem thời tiết dùng OpenWeatherMap API, xây dựng theo hướng kiến trúc rõ ràng và dễ mở rộng.

## Tính năng
- Xem thời tiết theo GPS
- Tìm kiếm theo thành phố
- Dự báo 5 ngày
- Hiển thị nhiệt độ, độ ẩm, gió, cảm giác thực

## Điểm nổi bật
- Kiến trúc MVVM, tách lớp rõ ràng
- Tích hợp REST API, xử lý JSON
- Quản lý trạng thái và dữ liệu forecast
- UI tối giản, responsive
- Xử lý permission và async network

## Công nghệ
- Kotlin / Java
- Android Studio
- OpenWeatherMap API
- Retrofit
